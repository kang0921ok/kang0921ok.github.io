## Gradle

### Gradle?
빌드 자동화 도구 중 하나로, 컴파일 및 링크와 같은 활동을 통해 [Artifact](https://en.wikipedia.org/wiki/Artifact_(software_development))를 생성합니다.  
Gradle 빌드 스크립트 Groovy, Kotlin DSL을 사용하여 작성 됩니다.
